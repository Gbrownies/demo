Git Commands

git init - Create new local repository

git status - Shows status of current commit

git add - Adds directory to staging area to commit

git add . - Adds everything in current directory to staging

git commit -m "{message}" - Commits the staged files into the repository

git log - View log of previous commits

git checkout -b {branch name} - Creates new branch with chosen name

git checkout {branch name} - Moves working directory to that branch

git branch - Shows all branches

git branch -D {branch name} - Deletes branch

git merge {branch name} - Merges the bracnh to active branch

git remote add {repo name} https://github.com/{github username}/{github repo name}.git - Creates remote repository linked to the specified repo

git push -u {repo name} {branch name} - Pushes current branch to specified branch in specified repository
